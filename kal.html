Tugas Determinan & Invers Matriks

A. Determinan Matriks

1. Matriks 2x2

A = | -7  -5 |
    |  1   4 |

Penjelasan:

Untuk matriks 2x2, determinan dihitung dengan rumus:

det(A) = (a × d) - (b × c)

Dimana:

  • a = -7
  • b = -5
  • c = 1
  • d = 4

Langkah perhitungan:

det(A) = (-7 × 4) - (-5 × 1)
       = -28 - (-5)
       = -28 + 5
       = -23

Kesimpulan: Determinan matriks adalah -23

2. Matriks 3x3

A = | 0  2  -3 |
    | 1 -2  -1 |
    | 0  0   1 |

Penjelasan:

Matriks ini merupakan matriks segitiga atas karena semua elemen di bawah diagonal utama bernilai 0.

Rumus determinan untuk matriks segitiga:

det(A) = hasil perkalian elemen diagonal utama

Langkah perhitungan:

det(A) = 0 × (-2) × 1
       = 0

Kesimpulan: Determinan matriks adalah 0

3. Matriks 4x4

A =
|  1  -3   1   1 |
| -3   1   1   1 |
|  1   1  -3   1 |
|  1   1   1  -3 |

Penjelasan:

Untuk matriks ukuran 4x4, determinan bisa dihitung dengan ekspansi baris atau eliminasi baris.

Dengan metode eliminasi, kita mengubah matriks menjadi bentuk segitiga. Setelah dilakukan operasi baris elementer, ditemukan bahwa terdapat baris yang saling bergantung (tidak independen).

Artinya matriks bersifat singular.

Kesimpulan:

det(A) = 0

B. Invers Matriks

Penjelasan:

Invers matriks hanya bisa dihitung jika determinan ≠ 0.

Rumus umum:

A⁻¹ = 1/det(A) × adj(A)

4. Matriks 2x2

A = | -7  -5 |
    |  1   4 |

Langkah 1: Hitung determinan

det(A) = (-7 × 4) - (-5 × 1)
       = -23

Langkah 2: Gunakan rumus invers 2x2

A⁻¹ = 1/(ad - bc) × | d  -b |
                   | -c  a |

Substitusi nilai:

A⁻¹ = 1/(-23) × | 4   5 |
                | -1 -7 |

Hasil akhir:

| -4/23  -5/23 |
|  1/23   7/23 |

Kesimpulan: Matriks memiliki invers karena determinan ≠ 0

5. Matriks 3x3

A = | 0  2  -3 |
    | 1 -2  -1 |
    | 0  0   1 |

Penjelasan:

Determinannya sudah dihitung sebelumnya yaitu 0.

Karena det(A) = 0, maka:

  • Matriks tidak memiliki invers
  • Matriks disebut matriks singular

6. Matriks 4x4

A =
|  1  -3   1   1 |
| -3   1   1   1 |
|  1   1  -3   1 |
|  1   1   1  -3 |

Penjelasan:

Determinannya = 0, sehingga matriks juga singular.

Kesimpulan:

  • Tidak memiliki invers

Kesimpulan Akhir

  • Determinan digunakan untuk menentukan apakah matriks memiliki invers
  • Jika det(A) ≠ 0 → matriks punya invers
  • Jika det(A) = 0 → matriks tidak punya invers
