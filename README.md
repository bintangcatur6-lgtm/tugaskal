# 📊 Tugas Determinan dan Invers Matriks

Website statis ini dibuat untuk memenuhi tugas praktikum mengenai perhitungan **determinan** dan **invers matriks** menggunakan metode ekspansi baris dan adjoin.

---

## 🌐 Demo Website

🔗 https://username.github.io/tugas-determinan/

---

## 📚 Deskripsi

Proyek ini berisi:

* Perhitungan determinan matriks (2x2, 3x3, 4x4)
* Perhitungan invers matriks menggunakan metode adjoin
* Penjelasan langkah-langkah perhitungan secara lengkap

---

## 🧮 Materi yang Dibahas

### 🔹 Determinan Matriks

* Rumus determinan 2x2
* Matriks segitiga (perkalian diagonal)
* Konsep matriks singular

### 🔹 Invers Matriks

* Rumus:

  A⁻¹ = 1/det(A) × adj(A)

* Syarat invers:

  * det(A) ≠ 0

## 🎯 Kesimpulan

* Determinan digunakan untuk menentukan apakah matriks memiliki invers
* Matriks dengan determinan 0 disebut **matriks singular**
* Hanya matriks dengan determinan ≠ 0 yang memiliki invers
